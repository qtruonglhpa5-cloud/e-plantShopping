# Paradise Nursery 🌿

Paradise Nursery is an online plant shopping web application built using React and Redux.

## Features
- Browse plants by categories (Aromatic, Medicinal, Indoor)
- View plant details (image, name, description, price)
- Add plants to cart
- Update cart quantities
- Remove items from cart
- View total price dynamically

## Technologies Used
- React
- Redux Toolkit
- CSS

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
